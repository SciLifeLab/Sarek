The Lumpy and Manta folder contains vcf files generated by Lumpy and Manta.
the TCGA calls were generated using the G15512.HCC1954.1.bam as tumor and HCC1954.NORMAL.30x.compare.bam as a normal sample.
the P2109_123 samples were generated using the CreateTranslocations pipeline.
the db files in the simulated_variants folder contain the the tumor and normal variants of the simulated sample.
